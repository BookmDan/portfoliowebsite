Project Clean and Go

Business Management Software

I designed and implemented a database from scratch for this program to utilize using MySQL workbench and MySQL server. This program is capable of managing all company data, utilizing java backend and MySQL queries. Originally this was accomplished on a localhost server, after creating a script to fill my data base with dummy data, I migrated the database to a AWS cloud server and connected by java program to that.

Designing the schema for this project took a long time and I learned a lot about MySQL, relations between keys and databases in general through that process. Connecting the java program to the database was my second step and helped me learn how programs interract, in the process of doing this I gained valuable experience connecting technologies together.

Now I am learning HTML/CSS and other technologies and languages to build a front end user experience for this. Currently in the process of creating a website utilizing HTML, CSS. The goal is to have a fully functional full stack project where users can input data into the database from the website and then retrieve their data from the AWS cloud server.
