Project Web Scraper

Bus Route Parser

For this project, the goal was to learn regular expressions, also known as regex. In order to develop a deep understanding of how regex works. I decided to use the JAVA regex library and implemented pattern and matcher objects to perform the scraping of the King County Transit page.

This was a very frustrating project at first. It took awhile to understand the intricacies of regex, and also quite a bit of help from Dr. Farag in her office hours. After many hours of reading through the raw code of the king county transit website I was able to discern the patterns I needed.

After figuring out the patterns I needed. I implemented data structures I would need and the various functions, helper functions and more that I would need in order to pull it off. The end result is that the program will ask for user input and determine if that input is valid. It handles edge cases by parsing the website for valid destinations. Once given a valid destination and bus stop, the program will calculate the time between each stop and give you an approximate time that your journey should take.
